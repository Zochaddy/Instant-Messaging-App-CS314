

**`src/App.jsx`** - This is where you write your app! Everything else is just setup.

### What Each File Does (You Can Ignore Most of These):

- **`src/App.jsx`** ← **EDIT THIS ONE!** This is your main app component
- **`src/main.jsx`** - Just starts your app (don't touch this)
- **`index.html`** - The HTML page (don't touch this)
- **`package.json`** - Lists what packages you're using (don't touch this)
- Everything else - Just configuration files (ignore them)

**Example:**
```jsx
function App() {
  return (
    <div>
      <h1>Hello!</h1>
    </div>
  )
}
```

That's it! The `<div>` and `<h1>` look like HTML, but they're actually JavaScript (called JSX).

### To Run Your App:

1. Open terminal in the `frontend` folder
2. Type: `npm run dev`
3. Open the URL it shows you (usually http://localhost:5173)
